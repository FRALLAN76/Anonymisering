## 1. GRUNDPRINCIPER

### 1.1 Omvänt skaderekvisit (OSL 26:1)

**Lagtexten:**
> "Sekretess gäller inom socialtjänsten för uppgift om en enskilds personliga förhållanden, om det inte står klart att uppgiften kan röjas utan att den enskilde eller någon närstående till denne lider men."

Inom socialtjänstsekretessen gäller **omvänt skaderekvisit** (stark sekretess):
- **Presumtion för sekretess** - uppgifter är som huvudregel sekretessbelagda
- **Bevisbörda**: Den som vill lämna ut måste bevisa att det är "klart" att utlämnande kan ske utan men
- **Vid tveksamhet**: MASKA ALLTID
- **Men** = olika former av integritet (inte bara ekonomisk skada)

**Praktisk konsekvens:**
Tjänstemannen kan ofta inte lämna ut uppgifter utan att känna till mottagarens identitet och syfte. Den som begär uppgifter kan behöva uppge vem hen är och varför hen vill ha uppgifterna.

**Jämförelse:**
| Typ | Huvudregel | Sekretess gäller om... |
|-----|------------|------------------------|
| **Omvänt** (stark) | Sekretess | ...det inte är **klart** att röjande är OK |
| **Rakt** (svag) | Offentlighet | ...det kan **antas** att röjande skadar |

### 1.2 Sekretessgränser
| Tidsgräns | Lagrum | Beskrivning |
|-----------|--------|-------------|
| 70 år | OSL 26:1 | Sekretess upphör efter 70 år |
| Livstid | - | Vissa uppgifter skyddas så länge personen lever |

**OBS:** Arkivet gör alltid sekretessprövning av handlingar yngre än 70 år.

### 1.3 Sekretessbrytande bestämmelser

Sekretessen kan brytas i följande fall:

| Lagrum | Situation | Beskrivning |
|--------|-----------|-------------|
| OSL 26:3 | Samtycke | Den enskilde kan helt eller delvis häva sekretessen |
| OSL 26:4 | Nödvändigt utlämnande | Vid fara för liv eller hälsa |
| OSL 26:6 | Partsinsyn | Part har rätt till insyn i eget ärende |
| OSL 25:12 | Barn/missbruk | Uppgifter om barn under 18 eller missbrukare kan lämnas till socialtjänsten |
| OSL 10:2 | Myndighetssamverkan | Nödvändigt för att myndigheten ska kunna fullgöra sin verksamhet |

## 3. TREDJE MAN - SÄRSKILT SKYDD

### 3.1 Definition
**Tredje man** = Person som nämns i akten men som inte är:
- Den som begär ut handlingarna (beställaren)
- Den som handlingarna primärt avser

**Rättslig grund:** OSL 10:3 reglerar konflikten mellan partsinsyn och tredje mans skydd.

### 3.2 Uppgifter om tredje man som ALLTID ska skyddas

| Uppgift | Känslighetsnivå | Lagrum | Regel |
|---------|-----------------|--------|-------|
| Namn | KRITISK | OSL 26:1 | Maska ALLTID |
| Kontaktuppgifter | KRITISK | OSL 26:1 | Maska ALLTID |
| Relation till enskilde | HÖG | OSL 26:1 | Maska om den avslöjar identitet |
| Innehåll i anmälan | KRITISK | OSL 26:1 | Maska ALLTID om anmälaren kan identifieras |

**Undantag (OSL 10:3):** Sekretess kan upprätthållas även mot part om det är av "synnerlig vikt" att uppgiften inte röjs - t.ex. om anmälaren riskerar hot eller repressalier.

### 3.3 Anmälare (orosanmälare) - särskilda regler

| Situation | Regel | Motivering |
|-----------|-------|------------|
| Professionell anmälare (polis, skola, BVC) | Myndighetsnamn kan vara OK, inte individnamn | Myndigheten är offentlig |
| Privatperson som anmälare | Maska ALLTID identitet | Skydd mot repressalier |
| "PRIVATPERSON" i text | Behåll - redan anonymiserat | Ingen identifiering möjlig |
| Anmälan som kan röja anmälare | Maska innehållet | Indirekt identifiering |

### 3.4 Vanliga tredje man-roller att identifiera

| Roll | Typiskt förekommande | Åtgärd |
|------|---------------------|--------|
| Mormor/morfar | Släkting i barnärende | MASKA |
| Farmor/farfar | Släkting i barnärende | MASKA |
| Granne | Vittne/anmälare | MASKA |
| Lärare/förskollärare | Professionell kontakt | MASKA namn, behåll roll |
| Tidigare partner | Nämnd i ärende | MASKA |
| Anmälare | Orosanmälan | MASKA |
| Barn till tredje man | Nämns indirekt | MASKA |

## 7. KONTEXTUELLA REGLER

### 7.1 Barn under 18 år
- **Förstärkt skydd** för alla uppgifter
- Barnets bästa ska alltid vägas in
- Vid tveksamhet: MASKA

### 7.2 Våldsutsatta personer
- **Särskilt skyddsbehov**
- Kontaktuppgifter är KRITISKA
- Uppgifter som kan leda till lokalisering: MASKA

### 7.3 Relationella risker
Bedöm om röjande kan leda till:
- Konflikt inom familjen
- Hot eller våld
- Försämrad relation mellan parter

### 7.4 LVU-ärenden - särskilda regler

**Lag (1990:52) med särskilda bestämmelser om vård av unga (LVU)**

| Situation | Regel | Motivering |
|-----------|-------|------------|
| Vårdnadshavare begär ut barnets akt | Partsinsyn gäller, MEN... | Vårdnadshavare är fortfarande part |
| Känsliga uppgifter om barnet | Kan hemlighållas vid "betydande men" | Barnets bästa |
| Uppgift som kan missbrukas | MASKA om risk för allvarlig skada | Skydd för barnet |
| Barn 13+ år motsätter sig utlämnande | Beakta barnets vilja | Mognadsbedömning |

**När kan uppgifter hemlighållas för vårdnadshavare?**
- Risk för att barnet skadas allvarligt (psykiskt, fysiskt)
- Speciella skäl tyder på att uppgiften kan missbrukas
- Hedersrelaterat förtryck eller våld i familjen
- Barnet självt motsätter sig (vid tillräcklig mognad)

**Viktigt:** Vårdnadshavare har rätt att få information om hur det går för barnet och att läsa anteckningar - men socialnämnden kan begränsa detta vid LVU-placering.

### 7.5 Sekretess mellan verksamhetsgrenar

**Inre sekretess:**
- Hälso- och sjukvård och socialtjänst är separata verksamhetsgrenar
- Sekretess gäller mellan dessa även inom samma kommun
- Personal får endast ta del av uppgifter som behövs för sina arbetsuppgifter
- Dokumentation ska hållas åtskild (SoL vs HSL)
